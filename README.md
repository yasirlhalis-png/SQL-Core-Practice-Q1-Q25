# SQL Core Practice (Q1–Q25)

## 📌 Project Overview

This repository contains 25 real-world SQL practice queries designed to strengthen core SQL concepts through hands-on practice.

The queries are based on practical business scenarios using tables such as Employees, Customers, Orders, and Products.

---

## 📚 Topics Covered

- SQL SELECT
- WHERE
- ORDER BY
- BETWEEN
- IN
- Aggregate Functions
- GROUP BY
- HAVING
- EXISTS
- NOT EXISTS
- ANY
- ALL
- CASE
- NULL Functions
- SELECT INTO
- INSERT INTO SELECT
- UNION

---

## 💻 What I Practiced

- Filtering records using different conditions
- Working with multiple tables
- Creating backup tables using SELECT INTO
- Copying data using INSERT INTO SELECT
- Using EXISTS and NOT EXISTS with subqueries
- Comparing values using ANY and ALL
- Categorizing data using CASE
- Handling NULL values with SQL NULL Functions
- Combining query results using UNION

During this project, I improved my understanding of writing SQL queries for real-world business scenarios.

I learned when to use EXISTS, ANY, ALL, CASE, UNION, SELECT INTO, and INSERT INTO SELECT. I also became more confident in writing subqueries and handling NULL values correctly.
